Web App · React / Vite / Supabase

The Lineup

A free, mobile-first MLB draft and team simulation web app built solo from scratch. Users draft a 26-man roster, simulate a full season via a PA outcome model calibrated on real OBP/SLG/AVG distributions, and compete on a global leaderboard.

  • Draft engine with position-aware logic and 700+ real MLB players
  • Simulation model based on empirical plate appearance outcomes
  • Multilingual support (EN / FR / ES) — matching the international baseball landscape
  • Supabase leaderboard, Formspree contact, Google Analytics integration
  • Personal Hall of Fame via localStorage
Stack: React, Vite, JavaScript, Supabase Year: 2024–2025
